# TangleNet
IOTA IXI Module to read web content from the Tangle

## How to use
Use curl:
```
curl http://localhost:14265/ -X POST -H "Content-Type: application/json" -H "X-IOTA-API-Version: 1" -d "{'command': 'TangleNet.view'}"
```
